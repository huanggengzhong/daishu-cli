# 袋鼠前端脚手架

## 特色

- 常用的目录结构（你可以在此基础上修改自己的目录结构）;
- vue.config.js（配置了别名，你可以自行修改和更多配置）;
- 命令安装路由和配置，命令同时支持路由的动态加载;
- 命令安装 vuex 和配置，同时有动态加载 module 模块;
- axios 网络请求已二次封装;

### 首先,在电脑里全局安装

```js
npm install daishu-cli -g
```

### 创建项目命令

创建 vue 项目

```js
daishu create 项目名 -f vue
```

创建 react 项目(后续支持)

```js
daishu create 项目名 -f vue
```

#### 安装依赖

```js
cd 项目名
npm install
```
